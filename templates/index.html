OBJECTIVE.

Our focus was on two key objectives as part of the ZS RWD&I Hackathon’20:

Building Capability

Create an application programming interface (API), that leverages an AI based classification model, to identify COVID-19 status using CT scan imaging data, with high accuracy.

Creating Scale

Develop an initial vision roadmap to expand ZS imaging analytics offerings, across different RWD sources imaging modalities and disease areas.

VISION.

EXISTING ASSET ON COVID PATIENT CLASSIFICATION.

  • ZS has been developing a binary classification model to identify COVID-19 positive patients using lung CT images
  • The latest model was trained on ~540 CT images belonging to two categories namely, COVID-19 positive & negative
  • The model has F-1 Score of 87.3%, AUC of 93.1% and Accuracy of 87.7%, as evaluated on ~200 test images
  • The model is up to the mark when compared to radiologists1 performance both in terms of sensitivity & specificity
  • This model was submitted in a biomedical imaging grand challenge1, ranking 8/328 global participants
  • The model is currently available in the form of pickled object and one needs to understand specific coding frameworks to be able to use the model

CREATING CAPABILITIES.

  • During the hackathon, we will be scaling the imaging AI capability of ZS, by building out a COVID-19 patient identification API
  • COVID-19 Classification API
    • Assess user requirements
    • Design UI to display required features
    • Enable user to upload CT images
    • Display Prediction
    • Display Grad-CAM2 image, etc.
    • Write a back-end code to implement the functions of required features
    • Link the COVID-19 classification model pipeline to the UI, to ingest, preprocess & predict uploaded images

PLAN TO SCALE.

  • COVID-19 Classification API
    • We plan to build an interactive UI where the users (radiologists, clinical teams, etc.) can upload the lung CT images & get real-time predictions
  • Image Classification Platform
    • This will be an extension of Phase-I where the users will be able to train their own image classification AI models across disease areas
  • Imaging AI solution
    • This will be an extension of Phase-II where capability will also include the model training for multi-class classification, image segmentation, disease progression, etc.
